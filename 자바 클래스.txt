<클래스와 객체>
- 자바 프로그램에서는 모든 것이 클래스를 통해서 작성됨.
- 클래스: 객체를 만드는 기본 도구. 어떤 객체를 표현하는 데이터와 객체의 행동을 나타내는 메소드로 구성됨.
예) 차를 객체화한다고 해봅시다.
	차를 나타내는 데이터를 속성이라고 함: 차종, 색, 연식 등
	차의 동작을 메소드라고 함: 달리기, 속도, 정차 등

- 객체와 인스턴스:
	클래스를 작성하면 그 클래스의 객체 또는 인스턴스를 생성할 수 있음.
	객체와 인스턴스는 의미적으로 약간의 차이가 있지만 혼용해서 사용해도 됨.
	인스턴스는 new 키워드를 이용하여 생성함.
- 속성과 메소드:
	Car 클래스에서 kind, color, year는 인스턴스의 데이터.
	즉 인스턴스를 특징지을 수 있는 각 인스턴스마다 갖는 데이터가 됨.

- 인스턴스를 생성하는 new 키워드
	클래스명 객체명 = new 클래스명();

- 생성자: 인스턴스가 생성되는 순간에 자동으로 호출되는 특별한 메소드로 인스턴스에 필요한 공간을 만들고 초기화하는 역할을 함.
- 특징
	1. 생성자명은 클래스명과 같아야 함.
	2. 생성자는 인스턴스가 생성될 때 자동으로 호출되는 특별한 메소드임.
	3. 생성자는 반환값이 없는 메소드임.
	4. 생성자는 얼마든지 오버로딩하여 사용할 수 있음.

- 인스턴스 변수와 클래스 변수: 
	클래스를 통해 인스턴스를 생성할 때, 각 인스턴스가 갖는 데이터 영역이 있고, 
	인스턴스들끼리 공유하는 데이터 영역이 있음.
	각 인스턴스가 갖는 데이터: 인스턴스 변수
	인스턴스들 간에 공유하는 데이터: 클래스 변수
	클래스 변수: 클래스당 한 개가 만들어져서 모든 인스턴스들이 공유하는 것.
		  static 키워드를 이용하여 만듬.
